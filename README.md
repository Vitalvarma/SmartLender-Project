# Smart Lender — Loan Eligibility Prediction

Smart Lender is a machine-learning-powered web application that predicts the
creditworthiness of loan applicants, helping banks and financial institutions
make faster, data-driven approval decisions. Four classification algorithms —
Decision Tree, Random Forest, K-Nearest Neighbors, and XGBoost — are trained
and compared on applicant data (income, credit history, education,
employment, dependents, and property area); the best performer is deployed
behind a Flask web interface for real-time predictions.

## Architecture

```
User Layer            Frontend Layer         Flask App Layer         ML Pipeline
─────────────         ───────────────         ───────────────        ──────────────────
Loan Applicant   →     home.html         →     Flask backend    →    Dataset collection
Credit Officer         predict.html            (app.py)               ↓
                       submit.html             route + validate      Preprocessing
                                                 + predict              (missing values,
                                                       ↓                 encoding, SMOTE,
                                                Prediction Engine        scaling)
                                                 loads rdf.pkl            ↓
                                                 + scale1.pkl          Train/test split
                                                       ↓                  ↓
                                                Approved / Rejected   Model training
                                                                       (DT, RF, KNN, XGBoost)
                                                                          ↓
                                                                       Evaluation & selection
                                                                       (best model → rdf.pkl)
```

## Data

The dataset is the classic loan-eligibility dataset (`Loan_ID`, `Gender`,
`Married`, `Dependents`, `Education`, `Self_Employed`, `ApplicantIncome`,
`CoapplicantIncome`, `LoanAmount`, `Loan_Amount_Term`, `Credit_History`,
`Property_Area`, `Loan_Status`) — 614 applicants, one row per loan
application.

## ML pipeline

1. **EDA** — count plots and distribution plots for categorical/numerical
   features; bivariate and multivariate views (e.g. gender × income ×
   approval) to spot patterns before modeling.
2. **Missing values** — categorical columns filled with the mode, numerical
   columns filled with the mean.
3. **Categorical encoding** — `Gender`, `Married`, `Education`,
   `Self_Employed`, `Property_Area`, `Loan_Status` mapped to integers;
   `Dependents` cleaned (`"3+"` → `3`) and cast to int.
4. **Class balancing** — SMOTE oversampling of the minority class so the
   models aren't biased toward the majority outcome.
5. **Feature scaling** — `StandardScaler` fit on the balanced feature set.
6. **Train/test split** — 67/33 split, `random_state=42`.
7. **Model training & evaluation** — Decision Tree, Random Forest, KNN and
   XGBoost, each scored on train accuracy, test accuracy, and 5-fold
   cross-validation.

### Results

| Model         | Train accuracy | Test accuracy | 5-fold CV mean |
|---------------|:--------------:|:--------------:|:--------------:|
| Decision Tree | 100.0%          | 78.5%           | 75.0%           |
| Random Forest | 100.0%          | 81.4%           | 84.1%           |
| KNN           | 85.8%           | 74.9%           | 78.4%           |
| **XGBoost**   | **96.5%**       | **81.0%**       | **82.1%**       |

**XGBoost** is deployed — it gives the best balance of generalization (a
smaller train/test gap than Decision Tree or Random Forest) and
cross-validated accuracy among the models tested. The trained model
(`rdf.pkl`) and the fitted scaler (`scale1.pkl`) are saved for reuse by the
Flask app.

## Running it yourself

```bash
git clone <this-repo-url>
cd SmartLender
pip install -r requirements.txt

# 1. Train the model (writes Flask/rdf.pkl and Flask/scale1.pkl)
cd Training
python train_model.py

# 2. Launch the web app
cd ../Flask
python app.py
```

Open the URL shown in the terminal (defaults to `http://127.0.0.1:5000/`),
click **Check eligibility**, fill in the applicant's details, and submit to
see the Approved/Rejected prediction with the model's confidence.
